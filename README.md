## Course 1: Supervised Machine Learning
### Week 1: Introduction to Machine Learning
* **What I learned today:** Machine Learning isn't just about robots; it's about using data to make predictions (like spam filters or medical diagnoses).
* **Key Definition:** Supervised Learning is when you teach the computer by giving it "right answers" to learn from.
* **Question for later:** How much data does a model need before it becomes "accurate"?
